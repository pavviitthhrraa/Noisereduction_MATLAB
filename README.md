# Noise Reduction in Audio Signals (MATLAB)

## 📌 Project Overview
This is my first MATLAB project exploring **Signals & Systems**.  
I implemented a simple **moving average filter** to reduce noise in an audio signal.

## 🎧 Steps
1. Load sample audio (`handel.mat`).
2. Add Gaussian noise.
3. Apply a moving average filter.
4. Compare original, noisy, and filtered signals.

## 🖼️ Results
- Plots show how filtering smooths the noisy signal.
- Audio playback demonstrates noise reduction.
